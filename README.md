# Hancli

Hanzi flashcards in the command line

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hancli'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hancli

## Usage
From this directory, a simple game:

	ruby -Ilib bin/han
